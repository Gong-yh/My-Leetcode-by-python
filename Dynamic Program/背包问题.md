# 背包
## 0-1背包
有n个物品，每个物品仅限一种
解法通常是构造二维数组dp_i,j or 一维滚动数组dp[i]
 要注意遍历顺序【二维顺序，一维倒序（防止重复添加）】
### M416 分割等和子集
```python
class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        sumAll = sum(nums)
        if sumAll % 2:
            return False
        target = sumAll // 2

        dp = [False] * (target + 1)
        dp[0] = True

        for i in range(len(nums)):
            for j in range(target, nums[i] - 1, -1):
                dp[j] = dp[j] or dp[j - nums[i]]
        # print(dp)
        return dp[-1]
```
